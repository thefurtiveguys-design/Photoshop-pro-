# Horror Poster Generator 🎬💀

A web application that generates professional horror movie posters in PSD format.

## Features
- 🎨 A4 size (2480×3508px) at 300 DPI
- 📁 Editable PSD with multiple layers
- 💀 Horror-themed design with blood effects
- 🌐 Web interface - no installation needed

## Installation

### Local Development
```bash
pip install -r requirements.txt
python app.py
